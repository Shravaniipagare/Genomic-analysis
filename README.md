# Genomic Sequence Analysis using Biopython

## Overview
Fetched and analyzed gene sequences from 5 organisms using NCBI Entrez and Biopython. Computed GC content, nucleotide frequency, and pairwise alignment scores. Visualized results using Matplotlib and Seaborn.

## Tools Used
Python | Biopython | Pandas | Matplotlib | Seaborn | NCBI Entrez | Google Colab

## What this project does
- Fetches DNA sequences from NCBI for 5 organisms (Human, Chimpanzee, Mouse, Rat, Zebrafish)
- - Computes GC content and nucleotide frequency distributions
- Performs pairwise alignment (Human vs other organisms)
- Visualizes results using Matplotlib and Seaborn bar charts
- Exports results to structured CSV files for downstream statistical analysis
- Solves 10+ Rosalind bioinformatics problems (FASTA parsing, RNA transcription, protein translation)

## Files
- `Genomic_Sequence_Analysis.ipynb` — main project notebook
- `gc_content.png` — GC content bar chart
- `nucleotide_frequency.png` — nucleotide frequency chart
- `gc_content_results.csv` — exported GC content results
- `nucleotide_frequency_results.csv` — exported nucleotide data
